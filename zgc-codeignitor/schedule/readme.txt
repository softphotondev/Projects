== Description ==
Rezervy is a great PHP script for various general requirements of each appointment based businesses. Rezervy provide bundle of features to make your work easy and smooth. Rezervy helps you to handle Categories-Services-Add-ons,  SEO meta tags,  Compatible with Strict Mode , Coupon discount, Recurring discounts, Tax/ Vat / GST,  Multi-currency supported, One page Checkout, Easy and smooth Admin panel, Customer dashboard,  Registered and Guest User Checkout,  Auto Confirm Appointments, Weekly Schedule,  Working & Off days management,  Registered and Guest Customers listing with booked appointment details,  Payment details,  Export all bookings, Customer information , Payment information and services information, Support tickets, Fully Responsive & much more we adding. == Installation ==
1. Upload the folder in your required directory & Set database credentials in config.php.
2. Run the directory URL in browser.
3. Verify purchase code and configure default company, payment, SMS and email settings.
